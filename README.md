## Preface

AAInfographics-Pro is a professional version of [AAInfographics](https://github.com/AAChartModel/AAChartKit-Swift) , it is an elegant and friendly chart framework for iOS, iPadOS, macOS. AAInfographics-Pro is a more powerful data visualization framework that supports more types beautiful chart like bellcurve, bullet, columnpyramid, cylinder, dependencywheel, heatmap, histogram, networkgraph, organization, packedbubble, pareto, sankey, series, solidgauge, streamgraph, sunburst, tilemap, timeline, treemap, variablepie, variwide, vector, venn, windbarb, wordcloud, xrange charts and so on.

## Requirements

- iOS 13.0+
- macOS 10.13+
- Swift 5
- Swift Package Manager tools 5.3+

## Installation

### CocoaPods

```ruby
pod 'AAInfographics-Pro'
```

Import the module in Swift with:

```swift
import AAInfographics_Pro
```

### Swift Package Manager

In Xcode, add the package dependency:

`https://github.com/AAChartModel/AAChartKit-Swift-Pro.git`

If you prefer a manifest-based setup, add:

```swift
dependencies: [
    .package(url: "https://github.com/AAChartModel/AAChartKit-Swift-Pro.git", from: "10.0.0")
]
```

Then link the `AAInfographics-Pro` product to your target and import the module in Swift with:

```swift
import AAInfographics_Pro
```

> Note: the Swift Package product name is `AAInfographics-Pro`, while the Swift module name is `AAInfographics_Pro` because Swift module imports cannot contain `-`.
